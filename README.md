# Delivery Seguro Web3

Aplicación descentralizada (DApp) desarrollada para la Evaluación Final Transversal de Fundamentos de Blockchain.

## Descripción
Sistema de Escrow para comercio electrónico. Retiene el pago de una compra en un Smart Contract hasta que un Oráculo (API de la empresa de despachos) confirma que el paquete fue entregado exitosamente al comprador.

## Tecnologías
* **Contratos Inteligentes:** Solidity.
* **Token:** ERC-20 (`MiToken`).
* **Frontend:** HTML, CSS, JavaScript (Ethers.js).
* **Wallet:** MetaMask.

## Instalación y Uso
1. Desplegar `MiToken.sol` y `Contrato.sol` en Remix.
2. Abrir `index.html` con Live Server.
3. Conectar MetaMask.
4. El comprador deposita los fondos.
5. Cambiar a la cuenta del Oráculo en MetaMask y confirmar la entrega.
